# 自然语言处理中文分词

## 方法概述

- 传统算法：使用N-gram，HMM，最大熵，CRF等实现中文分词
- 神经⽹络⽅法：CNN、Bi-LSTM、Transformer等
- 预训练语⾔模型⽅法：Bert等

## 数据集概述

- PKU 与 MSR 是 SIGHAN 于 2005 年组织的中⽂分词⽐赛 所⽤的数据集，也是学术界测试分词⼯具的标准数据集。
